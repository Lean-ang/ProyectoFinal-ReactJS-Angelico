# Leandro Angelico
## E-commerce realizado en React Js.

## Features
- Intercambiabilidad
- Modularizado
- Apto para cualquier dispositivo movil
- navegacion fluida

## Installation
Sonar requiere React.js  V18.2 para funcionar.

```sh
npm i
npm run dev
```
